# ⚡ config-generator

Generate full dev configs instantly

## About

Shipforge (formerly config-generator) is a production-ready SaaS platform that generates full developer stacks as downloadable ZIP projects. No setup hell—just select your stack and download.

## Features

- Vite + React frontend templates
- Express API backend templates
- Docker configurations
- CI/CD pipelines
- Prisma ORM setup
- JWT authentication boilerplate
- Environment validation

## Quick Start

Visit <a href="https://shipforge.dev">shipforge.dev</a> to generate your first stack.

## Local Development

If you want to run the generator locally:

1. Clone this repo
2. Follow the setup in the backend/ directory
3. Run the frontend

See backend/README.md for detailed setup.
