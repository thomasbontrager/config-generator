# Vite + React App

Production-ready Vite + React starter.

## Getting started

```bash
npm install
npm run dev
`
`
`
## Environment variables

1. Copy `.env.example` to `.env`
2. Update the values as needed

Example:
```env
VITE_APP_NAME=MyApp
VITE_APP_URL=http://localhost:3000
```
## Build for production
```bash
npm run build
```
